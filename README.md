# hsh
A shell in haskell

Has
  * Basic commands with arguments
  * Pipeing with >>=
  * Background with &
  * logic 
    * if then else 
    * && and ||
  * Built-ins
    * cd 
    * print var 
  * let
    * supports lambda functions and function definitions
